# Keymapper TUI

## Features
- Support for multiple keymaps
- Easy to use interface
- Easy shortcuts
- Uses the vial-qmk-keebart repository as the underlying firmware.
- Keymap visualizer
- Key picker with search and filter
- Key assignment to RGB color system: Select a color from a predefined palette or define a custom color.
    - Color systematic works like CSS with specifity: Individual color > Individual Category > Layer Category > Layer Color > default (unset)
    - Layer system with layer editor
    - Category system with category editor
    - Individual key customization -> key picker
    - Individual key color system and a color picker
- Keyboard picker based on keyboards supported in the vial-qmk-keebart repository
- Export and import layouts from simple markdown files
- Generate firmware in the TUI
- View build log
- Set output directory
- Set output filename
- Template system with template editor
- Template browser
- Metadata system with metadata editor
- Metadata browser
- Navigation inspired by VIM
- Help dialog
